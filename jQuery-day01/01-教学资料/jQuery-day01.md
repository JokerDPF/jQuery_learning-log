## 05-jQuery对象

(**ppt**)

hello，同学们好！

​		这一节咱们学习的是jQuery对象。


​		两者之前的语法虽然有类似的部分，但是可不能混着写哦，比如：通过jQ对象点出style属性，然后为backgroundColor赋值，或者通过dom对象调用css方法.

* trigger方法可以触发任意的事件，包括自定义的事件，什么叫做自定义事件呢？顾名思义就是事件名是咱们自己想的，写什么都可以

自定义事件是一种更为高阶的用法，咱们只需要了解基本的用法即可。

```javascript

// 2. trigger触发
.trigger('事件名')

// 4. 注册自定义事件
.on('自定义事件',function(){})

```
